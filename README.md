## 快速开始

导入以下文件

```HTML
    <script src="path/to/jquery.min.js"></script>
    <script src="path/to/jquery.snowflake.js"></script>
```

初始化

```JavaScript
    $(function() {
      $('body').snowflake();
    });
```
